# RFCs — Propuestas de cambio

Este directorio contiene las propuestas de cambio (Request for Comments) para modificar el reglamento.

## Flujo

1. Crear un archivo `NNNN-titulo-breve.md` con la propuesta.
2. Discutirla.
3. Si se aprueba, incorporar los cambios al reglamento y cerrar la RFC.
