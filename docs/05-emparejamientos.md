# 05 — Emparejamientos

## Sistema de emparejamiento

Los emparejamientos se realizan al inicio de cada ronda semanal.

### Criterios primarios

1. **Ascendencia**: se emparejarán jugadores con Ascendencia similar siempre que sea posible.
2. **Partidas previas**: se evitará repetir emparejamientos de rondas anteriores.
3. **Alianzas**: los jugadores aliados no se emparejarán entre sí a menos que no haya alternativa.

### Criterios secundarios

- **Disponibilidad**: se prioriza que todos los jugadores jueguen cada ronda.
- **Narrativa**: el Warmaster puede crear emparejamientos temáticos si refuerzan la historia de la campaña.

## Bye (descanso)

Si hay un número impar de jugadores, uno quedará descansando cada ronda. El jugador con bye obtiene **puntos de participación** pero no puntos de victoria.

## Flexibilidad narrativa

El Warmaster puede alterar los emparejamientos para favorecer la narrativa, siempre que lo comunique con antelación y lo justifique.
