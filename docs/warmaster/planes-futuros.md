# Planes futuros — Ideas para Fase II y III (solo Warmaster)

## Ideas pendientes

- **Localizaciones de Fase II y III:** definir nombres y conceptos para las zonas de combate de las fases posteriores.
- **Giros argumentales:**
  - ¿Qué pasa si los jugadores intentan destruir Vorago? ¿Es posible?
  - ¿Los necrones aparecen para reclamar su arma?
  - ¿Vorago puede ser contenido de nuevo? ¿A qué coste?
- **Eventos especiales:**
  - Tormenta disforme que aísla el sistema durante una semana.
  - Llegada de una facción neutral (Ej. Inquisición, Necrones Despiertos, Corsarios Eldar).
- **Personajes:**
  - ¿Crear un NPC que actúe como agente de Vorago (poseído/influenciado)?
  - El Portador de Vorago — un campeón que empieza a oír la voluntad del fragmento.

## Notas

- Rellenar a medida que avance la campaña.
- Anotar qué giros funcionan y cuáles no tras cada fase.
