# Liga Armageddon — Liga Narrativa de Warhammer 40,000

Reglamento completo para una liga narrativa de Warhammer 40,000 basada en **Dominatus (11.ª edición)**, pensado para clubes o asociaciones.

## Filosofía

La prioridad de la liga es la **narrativa**. El objetivo no es crear un entorno competitivo, sino representar la evolución de un conflicto militar.

Toda regla en este proyecto responde a estas preguntas:

- ¿Refuerza la narrativa?
- ¿Representa la escalada de una guerra?
- ¿Es sencilla de explicar?
- ¿Es coherente con Dominatus?
- ¿Evita listas abusivas sin perjudicar injustamente a una facción?
